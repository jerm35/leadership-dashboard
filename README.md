# Leadership Dashboard

Internal dashboard for team workload, risk tracking, and 1:1 support.

**Live:** https://jerm35.github.io/leadership-dashboard/

## Features
- Overview with KPI cards and escalation alerts
- People view with capacity tracking
- Workload balancing with task reassignment
- Project tracking with sort/filter by priority, risk, due date
- Risk log with aging tracker and overdue alerts
- 1:1 support with dev goals, coaching log, and meeting notes
- Decision log with rationale tracking

## Development
```bash
npm install
npm run dev
```

Pushes to `main` auto-deploy to GitHub Pages.
